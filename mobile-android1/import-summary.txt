Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => LaunchActivity/src/main/AndroidManifest.xml
* assets/ => LaunchActivity/src/main/assets/
* libs/android-support-v4.jar => LaunchActivity/libs/android-support-v4.jar
* res/ => LaunchActivity/src/main/res/
* src/ => LaunchActivity/src/main/java/

Old Build Tools:
----------------
The version of the build tools installed with your SDK is old. It
should be at least version 19.0.1 to work well with the Gradle build
system. To update it, open the Android SDK Manager, and install the
highest available version of Tools > Android SDK Build-tools.
